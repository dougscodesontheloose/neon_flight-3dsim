# Neon Flight Simulator ✈️ 

A browser-based 3D flight simulator built with Three.js. 

**Play it live here:** [INSERT_YOUR_GITHUB_PAGES_LINK_HERE]

## Features
*   **Dual Themes:** Toggle between **Synthwave** (Dark) and **Vaporwave** (Light) aesthetics.
*   **3 Aircraft Types:**
    *   **Vector Mk1:** Balanced Trainer.
    *   **Interceptor:** High speed, low lift.
    *   **Glider-X:** Slow speed, high lift biplane.
*   **Physics:** Basic aerodynamics including lift, gravity, drag, and banking.
*   **Visuals:** Infinite procedural city, retro sun, moving grid, and CRT scanline overlay.

## Controls
*   **W / S:** Throttle Control
*   **Arrow Keys:** Pitch and Bank
*   **Space:** Airbrake

## How to Run Locally
1. Clone this repository.
2. Because this project uses ES Modules (`import`), you need a local server.
3. If you have Python installed, run: `python -m http.server` inside the folder.
4. Open `http://localhost:8000`.
